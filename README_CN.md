## 1. Scallop

Scallop的数据结构定义如下：

- birth_time: Unix时间戳；
- raidus: 一个1到10的随机数，决定了扇贝的大小；
- card_color_index：卡牌背景色，随机从定义颜色集`ColorSet`中选取；
- main_color_index: 扇贝的主色，随机从预定义颜色集`ColorSet`中选取；
- id: 具有唯一性的扇贝编号，由`salt <- rand(1,100000)`，`id <- sha256(birth_time|raidus|main_color|salt)`生成；
